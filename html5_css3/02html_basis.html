html简介

html 即HyperText Markup Language：超文本标记语言，为文本添加语义，并且在浏览器中不会被显示（超文本/标签）；

html2 1995年W3C版本才有标准；2008年浏览器厂家连合W3C制定html5；

header标签

指定网站的标题/图标

添加网站的SEO相关信息（指定网站的关键字/指定网站的描述信息）

外挂一些外部的css/js文件

添加一些浏览器适配相关内容

a标签

github

假链接：href中写 # 或者 javascript: 区别为，#会返回到页面顶部
